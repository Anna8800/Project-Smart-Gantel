# Устройство для отслеживания упражнений с использованием TinyML


[![Platform: Arduino](https://img.shields.io/badge/Platform-Arduino-00979C)](https://www.arduino.cc/)
[![ML: TinyML](https://img.shields.io/badge/ML-TinyML-orange)](https://blog.tensorflow.org/2021/05/building-tinyml-application-with-tf-micro-and-sensiml.html)
[![App: Xamarin](https://img.shields.io/badge/App-Xamarin-3498db)](https://dotnet.microsoft.com/apps/xamarin)

> Прототип «умной гантели» с системой распознавания жестов на базе машинного обучения для микроконтроллеров (TinyML) и мобильным приложением для отслеживания прогресса.

---

## О проекте

Данный проект представляет собой прототип устройства для автоматического подсчёта повторений в силовых тренировках. Система распознаёт тип выполняемого упражнения с помощью гироскопического датчика и нейронной сети, работающей непосредственно на микроконтроллере (TinyML). Результаты передаются на смартфон через Bluetooth Low Energy и отображаются в мобильном приложении.

---

## Возможности

- Распознавание 3 типов упражнений:
  - Сгибание рук на бицепс
  - Жим над головой
  - Отведение рук в сторону
- Точность классификации: **98.7%** (на валидационной выборке)
- Низкое энергопотребление благодаря протоколу **BLE (Bluetooth Low Energy)**
- Кроссплатформенное мобильное приложение (Android / iOS)
- Защищённый разбираемый корпус для микроконтроллера
- Встроенный таймер тренировки
- Функции сброса результатов и завершения тренировки

##  Аппаратная часть

| Компонент | Описание |
|-----------|----------|
| **Микроконтроллер** | Arduino Nano 33 BLE Sense |

| **Связь** | Bluetooth Low Energy (BLE) |

## Машинное обучение (TinyML)

### Сбор и подготовка данных
1. Запись сигналов с гироскопа при выполнении каждого из 3 типов упражнений
2. Сбор данных от множества пользователей для обучения обобщённой модели
3. Предобработка сигналов и разметка по классам
4. Разделение датасета на обучающую и тестовую выборки

### Модель
- **Тип задачи**: Классификация временных рядов
- **Архитектура**: Нейронная сеть, оптимизированная для микроконтроллеров
- **Фреймворк**: TensorFlow Lite for Microcontrollers
- **Точность**: 98.7% на валидационной выборке
- **Порог уверенности**: 0.7 — при меньшей вероятности предсказание игнорируется

## Мобильное приложение

### Технологии
- **Платформа**: Xamarin (C#)
- **Поддержка**: Android и iOS из единой кодовой базы
- **Связь**: Bluetooth Low Energy (GATT-клиент)

### Интерфейс
| Экран | Функционал |
|-------|-----------|
| **Стартовый** | Подключение к устройству, переход к тренировке, выход из приложения |
| **Основной** | Подсчёт повторений по типам упражнений, таймер, кнопки «Сброс» / «Завершить» |

---

## Использование

1. Включите устройство переключателем на корпусе
2. Запустите мобильное приложение на смартфоне
3. В приложении нажмите **«Подключиться»** и выберите устройство `SmartDumbbell`
4. После успешного сопряжения перейдите к экрану тренировки
5. Выполняйте упражнения — повторения будут автоматически засчитываться
6. Используйте кнопки **«Сбросить»** или **«Завершить тренировку»** при необходимости

> **Важно**: Упражнение засчитывается только если уверенность нейросети ≥ 0.7. Это снижает количество ложных срабатываний.
